# DartStream Online Scoring App

Professional dart scoring application with real-time multiplayer support.

## Features

- ✅ Real-time multiplayer scoring
- ✅ Host/Away player system
- ✅ Live score synchronization
- ✅ Connection via 4-digit codes
- ✅ Turn-based control system
- ✅ Comprehensive match statistics
- ✅ Multiple game formats (301, 501, custom)
- ✅ Player library management
- ✅ Responsive portrait layout

## How to Use

### Single Player
1. Open the app
2. Select game format
3. Choose starting player
4. Start scoring

### Multiplayer
1. **Host**: Click "Connect" → Generate code
2. **Away**: Click "Connect" → Enter host's code
3. Wait for host to start the match
4. Take turns scoring automatically

## Local Development

1. Clone this repository
2. Run a local server:
   ```bash
   python -m http.server 8080
   ```
3. Open http://localhost:8080/index.html
